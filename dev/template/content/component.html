Project: Gulp Sample - with LESS

Overview

Front-end project to organize templates in components.


Built With

  • Bootstrap - Open source toolkit (v3.3.7)
  • jQuery - Dependency Management
  • LESS - Leaner Style Sheets

Installing

Please execute the commands below, to compile your project:

$ git clone
$ npm install
$ npm install --global gulp-cli
$ npm run dev


Note:

Be aware when updating NPM, as there may be a conflict of versions between LESS, npm for the operation of BrowserSync and construction of the task build.
